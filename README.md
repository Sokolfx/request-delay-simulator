# Request delay simulator

> A Request delay simulator.

## Features

- Nuxt ^2.15
- SPA or SSR

## Installation

- `composer install`
- Edit `.env` to set your `APP_HOST` and `APP_PORT` 
- `npm install`

## Usage

### Development

```bash
npm run dev
```

You can access your application at `http://{APP_HOST}:{APP_PORT}`.

### Production

```bash
npm run build
```
